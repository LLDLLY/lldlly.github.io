reselect

为什么我们需要reselect
Reselect 是一个 Redux 的选择器库：

  • Selector 可以计算衍生的数据，可以让 Redux 存储尽可能少的 state 。

  • Selector 非常高效，除非某个参数发生变化，否则不会发生计算过程。

  • Selector 是可组合的，它们可以输入、传递到其他的选择器。
